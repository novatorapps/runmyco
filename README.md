# RunMyCo, Inc.

RunMyCo builds and operates technology companies that help ambitious businesses grow, scale and succeed.

## Website

This repository contains the static corporate website for [RunMyCo](https://runmyco.com/). It is designed for deployment on Cloudflare Pages and does not require a build step.

## Local preview

Serve the repository root with any static HTTP server, then open the local URL in a browser.

```bash
python3 -m http.server 8000
```

## Deployment

The production website is hosted on Cloudflare Pages. The site can be deployed directly from the repository root.
